# Party Game Hub

Ứng dụng game di động nhiều người chơi (multiplayer) thời gian thực, cho phép các thiết bị kết nối và chơi cùng nhau trong mạng Wi-Fi cục bộ thông qua kết nối Peer-to-Peer (P2P).

## 🚀 Tính năng nổi bật
* **Local Multiplayer:** Kết nối người chơi cùng mạng Wi-Fi bằng giao thức P2P (mDNS/Bonjour và Sockets) mà không cần máy chủ internet.
* **Mini-Games:** Nhiều trò chơi nhỏ giải trí tích hợp.

## 🛠️ Công nghệ sử dụng
* **Flutter & Dart:** Xây dựng giao diện ứng dụng chéo nền tảng (Android & iOS).
* **Flame Engine:** Engine xử lý logic và đồ họa 2D trong game.
* **Provider:** Quản lý state của ứng dụng.
* **nsd (Network Service Discovery):** Khám phá thiết bị trong mạng LAN.
